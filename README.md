# Hyperfocus

Hyperfocus is a specialized web application designed to support individuals with ADHD by providing a structured, distraction-free environment for task management and hyperfocus sessions. 

The app features a visual timer, sequential task progression, randomized rewards, and customizable themes—all tailored to help you maintain momentum and stay on track without feeling overwhelmed.

## Features

- ⏱️ **Visual Timer:** A clear, high-contrast timer that helps visualize time remaining without inducing anxiety.
- 📝 **Sequential Task List:** Focus on one task at a time. The app guides you through your routine chronologically.
- ⏭️ **Flexible Progress:** Finished a task early? Skip to the next one instantly while keeping track of your actual time spent.
- 🎁 **Randomized Rewards:** Add short and long rewards. When you finish your session, the app picks one randomly to give you a dopamine boost!
- 🎨 **Customizable Themes:** Switch between calming Japandi, sleek Midnight, and high-energy Vibrant themes.
- 🔔 **Haptic & Audio Cues:** Choose from various completion sounds (Chime, Gong, Digital Beep, or Silent) and optional vibration support (Android only).
- 🔄 **Undo Actions:** Accidentally skipped or deleted a task? Easy undo options have your back.

## Getting Started

This project is built with React and Vite for a fast, modern web experience.

### Prerequisites
- Node.js installed on your machine

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/superfocus.git
   cd superfocus
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## Usage

1. **Create Tasks:** Add tasks you need to complete and estimate how long each will take.
2. **Start Session:** Click 'Start Session' to begin the countdown for your first task.
3. **Finish & Move On:** Let the timer finish naturally, or press the 'Fast Forward' button to complete a task early.
4. **Earn Rewards:** Once all tasks are done, enjoy a well-deserved, randomly selected reward!
